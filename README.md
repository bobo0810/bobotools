## 安装

```bash
稳定: pip install bobotools
新特性: pip install git+https://github.com/bobo0810/bobotools.git
```

## 工具类
### Pytorch
- 获取模型信息(模型大小、前向推理耗时等)

### 图像
- md5去重
- 读取在线图片(url/base64/file)
- 可视化bbox
- 验证图像完整性
- Tensor恢复为图片
### 文本
- 批量写入
- 批量读取

### 列表
- 分块，每块长度M
- 均分为N块
